# To-Do List App

This guide provides instructions for installing the To-Do List app on Windows 10, 11, and macOS. Follow the instructions specific to your operating system.

## Windows 11

1. **Ensure Python is installed:**
   - If not, download and install Python from [python.org](https://www.python.org/downloads/).

2. **Locate the "To-Do-List" folder:**
   - Find the folder in your downloads or saved location.

3. **Open in Terminal:**
   - Right-click the folder, select "Open in Terminal."

4. **Run the app:**
   - Copy and paste `python Main.py` into the Terminal or Command Prompt.
   - Press Enter, and the To-Do List app should start.

5. **Support and Assistance:**
   - For further assistance, check the support folder.
   - Real-time help is available on the Discord server: [To-Do List Discord Server](https://discord.gg/A6NaPSHnCU).

## Windows 10

1. **Ensure Python is installed:**
   - If not, download and install Python from [python.org](https://www.python.org/downloads/).

2. **Locate the "To-Do-List" folder:**
   - Find the folder in your downloads or saved location.

3. **Copy as path:**
   - Left-click the folder, hold down Shift, and select "Copy as path."

4. **Open Command Prompt:**
   - Search and select the Command Prompt.

5. **Navigate to the folder:**
   - Type `cd ` in the command prompt, right-click to paste the path, and press Enter.

6. **Run the app:**
   - Copy and paste `python Main.py` into the Command Prompt.
   - Press Enter, and the To-Do List app should start.

7. **Support and Assistance:**
   - For further assistance, check the support folder.
   - Real-time help is available on the Discord server: [To-Do List Discord Server](https://discord.gg/A6NaPSHnCU).

## macOS

1. **Ensure Python is installed:**

2. **Locate the "To-Do-List" folder:**
